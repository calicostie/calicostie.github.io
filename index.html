如何用hugo搭建个人博客

安装 环境：windows 将hugo.exe移动至/D/Software/hugo目录 将路径添加到path 启动终端，使用hugo version验证安装 创建站点 hugo new site quickstart 生成一个quickstart项目 选择主题 首先，从Github下载主题并添加至本地theme目录 cd quickstart git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 然后向站点配置文件config.toml添加此主题 echo 'theme = "ananke"' >> config.toml 写博客 使用new命令新建博客： hugo new posts/my-first-post.md 新文件的模板如下： --- title: "My First Post" date: 2019-03-26T08:47:11+01:00 draft: true --- 修改draft选项为true以应用草稿 预览 hugo server -D 即可在本地http://localhost:1313/预览 部署 在quickstar目录中新建.gitignore，写入/public/ 第一次发布需要配置public git init git remote add origin git@github.
